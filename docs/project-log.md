# Palette Desk Project Log

## 2026-06-22: 特集ページ運用方針

### 方針
- トップページには「特集」枠を表示する。
- PC版では左カラム最上部、スマホ版ではトップページ上部に表示する。
- 表示件数は最大3件。
- 表示する特集は `features.js` の登録内容からランダムで選ぶ。
- 特集見出しは `feature-list.html` へリンクする。

### ページ構成
- `feature-list.html`: 特集一覧ページ。
- `features/*.html`: 各特集ページ。
- 各特集ページには `TOP ＞ 特集一覧 ＞ ページ見出し` のパンくずを入れる。

### 追加手順
1. `features/` に特集ページHTMLを追加する。
2. `features.js` の `features` 配列に `slug`, `title`, `href`, `summary`, `tag` を追加する。
3. トップのランダム表示と特集一覧は `features.js` から自動反映する。
4. 公開前にローカルで `feature-list.html` と該当特集ページを確認する。

### デザインメモ
- 特集枠は白/薄グレーの交互背景で読みやすくする。
- 見出しやセクションにはPalette Deskの緑、木目系オレンジ、薄グレーをアクセントとして使う。
- 装飾は情報を邪魔しない範囲に抑える。
